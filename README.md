# wchen27-jsc370.github.io

This repository contains the final report and interactive visualizations for the JSC370 project on TTC Bus Delays. The project analyzes bus delay data to identify patterns and potential causes of delays, with the goal of improving transit efficiency.

This project is hosted at `https://wchen27-jsc370.github.io` and includes the following key files:
- `final.qmd`: The Quarto document for the final report, which includes an introduction, methodology, results, and conclusions.
- `interactive_visualizations.qmd`: The Quarto document containing interactive visualizations that complement the final report, allowing users to explore the data and findings in more depth.
- `final.pdf`: The PDF version of the final report, which can be downloaded for offline reading.

All data used in this project is available in the `data` directory. The API used for data retrieval is the Meteo weather API, which does not require authentication. A link to the API can be found here: `https://open-meteo.com`. The rest of the data is sourced from the City of Toronto's open data portal, which provides comprehensive information on bus routes, schedules, and delays. The data can be accessed at `https://open.toronto.ca/`.
